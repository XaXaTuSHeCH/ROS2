Название: man
Описание: Показывает справочные страницы (manual) для команд Linux
Синтаксис: man [команда]
Пример: man grep
Примечание: Позволяет узнать все опции и описание команды. Для выхода нажмите Q. Можно искать по ключевому слову через /слово.
===
Название: cp
Описание: Копирует файлы и каталоги
Синтаксис: cp [опции] источник назначение
Пример: cp -r /home/user/docs /home/user/backup/
Примечание: -r копирует каталоги рекурсивно, -i спрашивает подтверждение при перезаписи
===
Название: mv
Описание: Перемещает или переименовывает файлы
Синтаксис: mv [опции] источник назначение
Пример: mv oldname.txt newname.txt
Примечание: Также используется для переноса файлов в другой каталог
===
Название: rm
Описание: Удаляет файлы и каталоги
Синтаксис: rm [опции] [файл/каталог]
Пример: rm -rf /home/user/temp/
Примечание: -r удаляет рекурсивно, -f отключает подтверждение (использовать осторожно!)
===
Название: cat
Описание: Выводит содержимое файлов
Синтаксис: cat [файл]
Пример: cat /etc/os-release
Примечание: Можно объединять файлы: cat file1 file2 > merged.txt
===
Название: grep
Описание: Ищет строки по шаблону в файлах
Синтаксис: grep [опции] "шаблон" [файлы]
Пример: grep -i "ros2" *.txt
Примечание: -i игнорирует регистр, -r ищет рекурсивно по каталогам
===
Название: find
Описание: Ищет файлы и каталоги по имени, размеру и другим критериям
Синтаксис: find [путь] [опции]
Пример: find /home/user -name "*.py"
Примечание: Можно искать по размеру (-size), дате изменения (-mtime) и владельцу (-user)
===
Название: head
Описание: Показывает первые строки файла
Синтаксис: head [опции] [файл]
Пример: head -n 5 logfile.txt
Примечание: -n задаёт количество строк (по умолчанию 10)
===
Название: tail
Описание: Показывает последние строки файла (обычно для логов)
Синтаксис: tail [опции] [файл]
Пример: tail -f /var/log/syslog
Примечание: -f “следит” за обновлением файла в реальном времени
===
Название: uname
Описание: Показывает информацию о системе
Синтаксис: uname [опции]
Пример: uname -a
Примечание: Отображает ядро, архитектуру и версию ОС
